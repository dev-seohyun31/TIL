# 실제 개념
## 전하와 전기 
### 1. 원자와 전하
원자는 양성자(Proton) + 중성자(Neutron) 로 이루어진 원자핵과, 그 주변에 확률적으로 분포하는 전자(Electron cloud) 로 구성됩니다.
전하는 입자가 가지는 물리적 속성(Physical property) 입니다.
| 입자  | 전하   |
| --- | ---- |
| 전자  | 음(-) |
| 양성자 | 양(+) |
| 중성자 | 0    |

일반적으로 고체 물질에서 양성자와 중성자는 격자에 고정되어 있고, 전자만 이동가능합니다. 따라서, 전류는 전자의 이동으로 형성됩니다. 

### 2. 전하량 보존 법칙
전하량(Q)의 단위는 C (Coulomb)으로, 닫힌 계에서 전체 전하량의 합은 생성되거나 소멸되지 않습니다. 
즉, 개별 물체의 전하량은 이동에 의해 변할 수 있지만, 전체 합은 항상 일정합니다. 
* `1 C = 약 6.242 × 10¹⁸ 개(624경 개)의 전자`

### 3. 도체 / 반도체 / 절연체
전자가 얼마나 자유롭게 이동할 수 있는지에 따라 물체를 3가지로 구분할 수 있습니다. 
| 물질  | 특징                   |
| --- | -------------------- |
| 도체  | 자유전자가 많아서 쉽게 이동      |
| 반도체 | 조건부로 이동 가능              |
| 절연체 | 전자 강하게 결합하여, 이동 거의 없음 |

따라서, 도체가 전기가 잘 통하는 이유는, 전하량이 많아서가 아니라 이동 가능한 전자의 비율이 높기 때문입니다.


## 전기 4요소
먼저, 개념을 다음과 같이 크게 볼 수 있습니다. 
* 전압 (Voltage, V): 전하를 움직이게 만드는 에너지 차이 (원인)
* 저항 (Resistence, R): 전류의 흐름을 방해하는 조건 (조건)
* 전류 (Current, I): 실제로 흐르는 전하의 이동 (결과)
* 전력 (Power, P): 실제로 소비되는 에너지 크기 (결과의 크기)

### 1. 전압 (Voltage, V)
전압이란 단위 전하당 에너지입니다: `V = W / Q`

전압은 항상 차이(전위차)로만 의미가 있습니다.
회로 관점에서 보면, 전원은 전압을 제공하고, 전압은 전류 흐름의 조건입니다. 
따라서, 전압이 순간적으로 떨어지면 Brown-out 현상이 일어나 MCU가 리셋(POR, BOR)이 될 수 있습니다.

### 2. 전류 (Current, I)
전류는 전하의 이동률입니다: `I = Q / t`

1A의 의미는 1초에 1C 이동했다는 의미를 가지고 있으며, 보통 `1 C = 약 6.242 × 10¹⁸ 개(624경 개)의 전자`입니다. 
전자는 -에서 +로 이동하지만, 관습적으로 전류는 +에서 -로 이동한다고 표현합니다. 


### 3. 저항 (Resistence, R)
저항은 전류 흐름을 방해하는 정도입니다. 
발생 원인으로는 격자 진동(온도), 불순물, 구조, 길이 및 굵기가 있을 수 있습니다. 
수식으로는 다음과 같이 표현됩니다: `R = p * L / A`

회로에서 저항은 전압 강하를 만들며, 전기 에너지를 열로 변환시킵니다. 
주로 전류 제한(LED), 전압 분배, Pull-up/Pull-down, 바이어스, 신호 형상을 위해 사용됩니다.

### 4. 전력 (Power, P)
전력은 에너지의 소비 속도입니다. : `P = E / t`

E는 에너지의 양으로, 1J은 1C 전하를 1V 차이로 이동시키는 에너지입니다. : `E = V * Q`

또한, 다음 아래와 같이 변형되어 해석될 수 있으며, 전력 대부분은 열로 변환되어 발열을 일으킵니다.
```
P = V · I
P = I² · R
P = V² / R
```


### 옴의 법칙
옴의 법칙이란 도체(선형 저항)에서 전류는 전압에 비례하고, 저항에 반비례한다는 정리입니다: `V = IR`

하지만 LED, 다이오드, 트랜지스터는 비선형 소자이기 때문에 저항과 함께 사용해야 전류 제어가 가능해집니다.

#### 예제 1. LED 저항 계산
 GPIO = 3.3V, LED Vf = 2V, 목표전류 = 10mA일 경우, 남는 전압은 1.3V이므로 R = 130옴 을 계산하여 LED 저항값을 설정할 수 있습니다.
#### 예제 2. GPIO 보호
GPIO 최대 전류 = 20mA, 전원 = 3.3V일 경우, R이 0옴에 가까워지면, I는 무한대로 가까워지므로 핀 손상 가능성이 매우 큽니다. 따라서 GPIO는 외부 저항 또는 드라이버와 함께 사용합니다.
